Thanks for installing Perplexity Shortcut

To use this extension, please set the keyboard shortcut to work globally:

  1. Go to chrome://extensions/shortcuts by clicking on it or typing it in the address bar
  2. Find "Perplexity Shortcut" in the list
  3. Click on the dropdown next to "Toggle Perplexity search input"
  4. Select Global from the options

Once you've done this, you can use Alt+Q (or your chosen shortcut) to quickly search Perplexity from any webpage!

Note: You may need to reload your browser or individual pages for the extension to start working after installation.
